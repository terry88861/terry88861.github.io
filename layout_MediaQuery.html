本文主旨與學習資源

本文的主要目標是說明如何運用 Media Queries CSS style property 屬性，來因應使用者在各種不同螢幕大小的終端設備( 例如: 桌機，筆電，平板，手機等 )開啟你的網頁時，而引發需要對網頁中的元素或佈局所做的調整。如果你想達到當使用者改變瀏覽器 browser size 大小時，網頁中的元素或佈局能隨之調整，你可以運用 Media Queries 來進行設計。

      Today people are using all sorts of devices that have different resolutions you have people using desktops, laptops, tablets and mobile phones and basically what this means is your webpage has to fit on all these devices.
